Detailed Description

The temperature beacon structure.

This defines the structure of the temperature beacon.

The temperature beacon consists of three members: the type of beacon (type), the decimal identifier (deci), and the temperature data (temperatureC[]). A diagram of the struct is included below.

    1 byte    4 bytes      4*X bytes 
+--------+----------+-----------------+
|  type  |   deci   |  temperatureC[] |
+--------+----------+-----------------+
    (Note: X = ARTEMIS_TEMP_SENSOR_COUNT)

◆ deci

uint32_t Artemis::DevicesAndSensors::temperaturebeacon::deci = 0

A decimal identifier for the beacon.

This value is the current uptime of the Teensy in milliseconds. By default, it is zero.

◆ temperatureC

float Artemis::DevicesAndSensors::temperaturebeacon::temperatureC[ARTEMIS_TEMP_SENSOR_COUNT]

The temperature data.

The temperature values are stored as an array of floating-point values. The length of the array is defined by the ARTEMIS_TEMP_SENSOR_COUNT constant.
