int32_t Artemis::Devices::read_current ( uint32_t  uptime)

Polls current sensors and transmits results.

This function iterates over the current sensors and packages the result as the payload of two packets. Both of these packets are added to the RFM23 and Astrodev queues for transmission.

Parameters
uptimeThe time, in milliseconds, since poweron.
Returns
int32_t Always returns 0.

int32_t Artemis::Devices::read_imu ( uint32_t  uptime)

Polls IMU and transmits results.

This function polls the IMU for accelerometer, gyroscope, and temperature data. The reply is packaged as the payload of a packet. This packet is then added to both the RFM23 and Astrodev queues for transmission.

Parameters
uptimeThe time, in milliseconds, since poweron.

Maps current sensor names to object instances.

This mapping correlates instances of the INA219 current sensor object with a string naming which part of the satellite the sensor is measuring. Each instance of the INA219 object is initialized with the I2C address of that sensor.
