Detailed Description

The payload size/radio status field.

This field serves two purposes:

  • For packets going from the Teensy to the radio, it holds the high byte of the payload size value.
  • For packets going from the radio to the Teensy, it holds a reply byte containing data about the radio's status.

The high byte of the payload size value.

The Most Significant Byte (MSB) of the size is first, and only the second byte is actually used for a maximum payload size of 255 bytes.The bytes are flipped, be careful.
