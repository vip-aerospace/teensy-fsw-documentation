Detailed Description

Definitions of Artemis Beacons.

This defines the beacons used in the packet communications protocol.

A 'beacon' defines the structure of data stored in a packet. It is stored as the payload of a packet, after that packet's header information. We do this to ensure that the data in a packet is always in a predictable location.

All Artemis beacon structs are packed structs, meaning there is no padding between their members. This ensures the packets are as small as possible.
